Logpile

Another framework for spying the errors of your application.

Download this project as a .zip file
Features : Main Server Client Vertx mods  

Vertx module

The server implementation use vertx for the events collects and the web console. Two modules were created for the treatment :

  • Mod-couchbase : A module persistor for the couchbase database.
  • Mod-jmx : A module which allow you to expose attribute or method in the jmx console.
